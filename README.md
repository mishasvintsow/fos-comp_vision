# Методический комплект дисциплины «Методы компьютерного зрения»

Репозиторий содержит рабочую программу дисциплины, фонд оценочных средств, контрольно-измерительные материалы и методические указания.

Дисциплина включает четыре модуля, 36 часов лекций и 36 часов лабораторных работ. Общая трудоёмкость составляет 144 часа (4 ЗЕТ), включая 18 часов самостоятельной работы и 54 часа проектной работы.

## Состав комплекта

Комплект включает РПД, модель оценивания, девять лабораторных работ, примеры тестовых заданий, универсальный проектный кейс, матрицу измеримости, указания обучающемуся и преподавателю и информационные ресурсы.

Организационные реквизиты конкретной реализации — код дисциплины, образовательная программа, университет, составители, индустриальный партнёр и конкретный проектный кейс — заполняются университетом-реализатором.
