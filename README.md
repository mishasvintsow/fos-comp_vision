# Методический комплект дисциплины «Методы компьютерного зрения»

Репозиторий содержит рабочую программу дисциплины, фонд оценочных средств, контрольно-измерительные материалы и методические указания.

Канонической основой тематического плана является редакция `РПД Компьютерное зрение_v2.xlsx`: четыре модуля, 36 часов лекций и 36 часов лабораторных работ. Общая трудоёмкость дисциплины составляет 144 часа (4 ЗЕТ), включая 18 часов самостоятельной работы и 54 часа проектной работы.

## Статус комплекта

Содержательная часть приведена к единой системе результатов и оценивания. `DL-1.2` заменён на `DL-1.4`; для генеративных моделей добавлены `DL-2.1` и `DL-2.2`. ЛР № 6 по YOLO измеряет `DL-3.1`, а `DL-3.3` среднего уровня и `PL-1.3` измеряются в ЛР № 9 через построение, испытание и обработку данных видеопайплайна.

Решением разработчиков программы предусмотренная исходным XLSX лабораторная работа по базовому видеопайплайну заменена ЛР № 8 по диффузионным моделям. Все девять лабораторных работ, перечисленных в РПД, обеспечены готовыми методическими материалами.

Организационные реквизиты конкретной реализации — код дисциплины, образовательная программа, университет, составители, индустриальный партнёр и конкретный проектный кейс — заполняются университетом-реализатором.
