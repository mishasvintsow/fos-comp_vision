# Ключи к примерам тестовых заданий

Ключи хранятся рядом с примерами, поскольку банк предназначен для разработки фактических вариантов, а не для закрытого тестирования. Номер вопроса действует внутри указанного тематического блока. Перед применением реализатор проверяет формулировки, ключи и соответствие фактическому содержанию курса.

## Блок 2. Диффузионные модели

**Нормативная привязка:** модуль 4, `DL-2.1`, средний уровень.

- вопрос 1: B — прямой и обратный процессы DDPM задаются дискретной марковской цепью;
- вопрос 2: C — в классической постановке сеть обучается предсказывать гауссов шум \(\epsilon\), добавленный на выбранном шаге;
- вопрос 3: B — VAE кодирует изображение в латентное пространство и декодирует результат обратно;
- вопрос 4: B — благодаря свойствам гауссовского процесса \(x_t\) можно получить непосредственно из \(x_0\) для выбранного шага \(t\);
- вопрос 5: C — слишком высокий CFG чрезмерно усиливает обусловленное текстом направление и может приводить к перенасыщению и геометрическим артефактам;
- вопрос 6: B — увеличение числа шагов существенно повышает время генерации, тогда как качество обычно выходит на плато и не обязано улучшаться;
- вопрос 7: B — cross-attention связывает текстовые представления с признаками, обрабатываемыми U-Net;
- вопрос 8: C — ControlNet добавляет к диффузионной модели обучаемое пространственное условие, например контуры, глубину или позу;
- вопрос 9: C — при малом `strength` добавляется мало шума, поэтому результат остаётся близким к исходному изображению;
- вопрос 10: A — negative prompt задаёт нежелательное условие, от которого механизм classifier-free guidance направляет генерацию.
