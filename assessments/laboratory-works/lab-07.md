- выполните загрузку  и предобработку данных из набора MNIST:

```python
x_train = (x_train.astype(np.float32) - 127.5) / 127.5
```

- далее изображение преобразуется в одномерный массив из 784 элементов:

```python
x_train = x_train.reshape(x_train.shape[0], 784)
```

- Определение параметров нейронной сети:

```python
random_dim = 100
adam = Adam(lr=0.0002, beta_1=0.5)
```

Здесь, random_dim задает размерность входного шума для генератора,  
adam – это оптимизатор с предустановленными параметрами.

- Создание архитектуры генератора: генератор принимает на вход вектор случайного шума и преобразует его в изображение.

```python
def create_generator(): generator = Sequential()
generator.add(Dense(256, input_dim=random_dim, kernel_initializer=initializers.RandomNormal(stddev=0.02)))
generator.add(LeakyReLU(0.2)) generator.add(BatchNormalization(momentum=0.8)) generator.add(Dense(512)) generator.add(LeakyReLU(0.2)) generator.add(BatchNormalization(momentum=0.8)) generator.add(Dense(1024)) generator.add(LeakyReLU(0.2)) generator.add(BatchNormalization(momentum=0.8)) generator.add(Dense(784, activation='tanh')) generator.compile(loss='binary_crossentropy', opti-
mizer=adam)
return generator
```

Здесь инициализируется слой с 256 нейронами. Важным моментом является инициализация весов с помощью нормального распределения со стандартным отклонением 0,02 – это может помочь в начальной фазе обучения.

LeakyReLU – это вариация функции активации ReLU, которая позволяет небольшим отрицательным значениям пройти через нейрон, предотвращая "затухание" градиентов.

Последний слой генератора использует функцию активации tanh, что гарантирует, что выходные значения будут в диапазоне [-1, 1].

- Создание архитектуры дискриминатора:

```python
def create_discriminator(): discriminator = Sequential()
discriminator.add(Dense(1024, input_dim=784, kernel_initializer=initializers.RandomNormal(stddev=0.02)))
discriminator.add(LeakyReLU(0.2)) discriminator.add(Dense(512)) discriminator.add(LeakyReLU(0.2)) discriminator.add(Dense(256)) discriminator.add(LeakyReLU(0.2)) discriminator.add(Dense(1, activation='sigmoid')) discriminator.compile(loss='binary_crossentropy', optimizer=adam)
return discriminator
```

Дискриминатор рассматривает одномерные массивы (изображения) длиной 784 пикселя и классифицирует их как реальные или сгенерированные. На последнем слое используется активация "sigmoid", что позволяет получать вероятность принадлежности изображения к реальному классу.

- Создание GAN:

```python
def create_gan(discriminator, generator): discriminator.trainable = False gan_input = Input(shape=(random_dim,)) x = generator(gan_input)
gan_output = discriminator(x)
gan = Model(gan_input, gan_output) gan.compile(loss='binary_crossentropy', optimizer=adam)
return gan
```

Эта функция соединяет генератор и дискриминатор в единую модель. При этом обучение дискриминатора отключается, чтобы обеспечить обучение только генератора при тренировке GAN.

- Обучение модели на наборе MNIST в течение 2 эпох с размером батча 128:

```python
def train_gan(epochs=1, batch_size=128): 
batch_count = x_train.shape[0] // batch_size 
generator = create_generator() 
discriminator = create_discriminator()
gan = create_gan(discriminator, generator) 
for e in range(epochs):
for _ in range(batch_count):
noise = np.random.normal(0, 1, size=[batch_size, random_dim])
generated_images = generator.predict(noise) 
image_batch = x_train[np.random.randint(0,x_train.shape[0], size=batch_size)]
X = np.concatenate([image_batch, gener-ated_images])
y_dis = np.zeros(2 * batch_size) y_dis[:batch_size] = 0.9
d_loss = discriminator.train_on_batch(X,y_dis)
noise = np.random.normal(0, 1,size=[batch_size, random_dim])
y_gen = np.ones(batch_size)
g_loss = gan.train_on_batch(noise, y_gen) print(f"Epoch {e} | Discriminator Loss: {d_loss}
| Generator Loss: {g_loss}") plot_generated_images(generator)
return generator, discriminator
```

Функция train_gan принимает два аргумента: epochs и batch_size, которые определяют количество эпох обучения и размер обучающего набора соответственно.

`batch_count = x_train.shape[0] // batch_size`: Эта строка определяет, сколько наборов для обучения в одной эпохе.

Затем создаются три модели: генератор, дискриминатор и GAN, объединяющая их вместе.

Внешний цикл `for e in range(epochs):` итерируется по эпохам обучения.

Внутренний цикл `for _ in range(batch_count):` итерируется по батчам в текущей эпохе. Для каждого набора:

- генерируется шумовой вектор noise, который передается в генератор для создания искусственных изображений;
- из обучающего датасета x_train случайным образом выбирается, реальный набор обучения, состоящий из изображений;
- сгенерированные и реальные изображения объединяются в один обуающий набор X;
- создается метка y_dis для обучения дискриминатора: первая половина имеет значение 0.9 (для реальных изображений), вторая половина – 0 (для искусственных изображений). Значение 0,9 используется вместо 1 для техники "label smoothing", что помогает стабилизировать обучение;
- дискриминатор обучается на наборе из реальных и сгенерированных изображений. Затем создается новый шумовой вектор, и генератор пытается обмануть дискриминатор через модель GAN.

После завершения каждой эпохи выводятся потери дискриминатора и генератора, а после завершения всех эпох отображаются примеры изображений, сгенерированных текущим генератором с помощью функции plot_generated_images.

В конце функции возвращаются обученные модели генератора и дискриминатора.

**Функция plot_generated_images:**

Эта функция предназначена для визуализации изображений, которые генерирует генератор сети GAN.

`generator`: модель генератора, которая принимает случайный шум и возвращает сгенерированные изображения.

`examples`: количество изображений, которые нужно сгенерировать и отобразить.

`dim`: кортеж, определяющий размерность сетки для отображения изобра-жений. По умолчанию отображает изображения в одну строку и 10 столбцов.

`figsize`: кортеж, определяющий физический размер фигуры (ширина x высота).

В функции выполняются следующие шаги:

- генерация случайного шума (noise) с помощью нормального распределения;

- создание фигуры с заданным размером figsize;
- в цикле отображение каждого сгенерированного изображения на сетке.
- отображение всей сетки сгенерированных изображений. Обучение GAN:
