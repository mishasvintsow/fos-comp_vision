# Фонд оценочных средств

| Раздел | Содержимое |
|---|---|
| [Модель оценивания](grading-policy.md) | формула `30/20/50`, пороги, шкала, правила теста, проекта и пересдачи |
| [Лабораторные работы](laboratory-works/README.md) | девять практических КИМ с отчётом и защитой |
| [Тест](tests/README.md) | банк вопросов, ключи и спецификация варианта |
| [Проектный кейс](project/README.md) | универсальный шаблон промежуточной аттестации |
| [Матрица измеримости](../docs/measurement-matrix.md) | связь индикаторов, уровней, заданий, артефактов и критериев |

Текущая аттестация включает лабораторные работы и тест. Промежуточная аттестация проводится в форме защиты проектной работы. Университет-реализатор указывает официальное название вида аттестации в соответствии с локальными нормативными актами.
