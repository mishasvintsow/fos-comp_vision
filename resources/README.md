Для универсальной версии комплекта этого перечня достаточно как библиографической основы. Перед реальным проведением дисциплины университет-реализатор дополнительно фиксирует только операционно значимые сведения: поддерживаемые версии Python и библиотек, используемые модели и наборы данных, условия доступа к ним и доступную замену при недоступности обязательного ресурса. Полная паспортизация каждой библиографической позиции не требуется.
