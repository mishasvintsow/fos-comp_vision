# Методические указания

Указания обучающимся включены непосредственно в [лабораторные работы](../assessments/laboratory-works/README.md) и [универсальный шаблон проектного кейса](../assessments/project/README.md). Общие правила кратко собраны в [памятке обучающемуся](students.md).

Основные регламенты:

- [модель оценивания, пороги и пересдача](../assessments/grading-policy.md);
- [матрица измеримости](../docs/measurement-matrix.md);
- [общие указания преподавателю по проведению и оцениванию ЛР № 1–9](teachers.md);
- [чек-лист преподавателя для защиты ЛР № 8](teachers/lab-08-code-review-checklist.md).

Университет-реализатор дополняет комплект локальными правилами подачи работ, апелляции, академической добросовестности и использования генеративного ИИ. Эти правила не должны изменять опубликованные критерии после выдачи задания.
